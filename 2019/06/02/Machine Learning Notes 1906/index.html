Formula Derivatives

Proof. \(ln(1-x) = -\sum^{\infty}{n=1}\frac{x^n}{n}\)
$$
\frac{d}{dx}ln(1-x) = \frac{-1}{ln(1-x)} \
-(1-x)^{-1} = -(1 + x + x^2 …) \
$$
$$
\begin{align}
ln(1-x) &= \int -(1 + x + x^2 …) \
&= -(x + \frac{x^2}{2} + \frac{x^3}{3} …) \
&= -\sum^{\infty}{n=1}\frac{x^n}{n}
\end{align}
$$
